- MVVM
- Clean Architecture

## 使用ツール

- Ktlint

## 設計パターン
- UDF (単方向データフロー)

## アプリを使用する際の注意点

Supabaseを使用するので、local.propertyファイルに以下のコードを記述してください。

SUPABASE_URL=https://twxdngcbimgolvfbglpk.supabase.co
SUPABASE_PUBLIC_KEY= ここにこれを記載するの怖いので、Twitter(新X)のDMでキープリーズみたいな感じで言ってくだされば、渡します。(@segnities007)
